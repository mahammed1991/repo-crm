RLSA Limitations and Boundaries (New)

  • We only help set-up RLSA on enabled AdGroups of enabled Campaigns OR all (enabled and paused) AdGroups of all (enabled and paused) Campaigns.
  • We can only apply RLSA to all AdGroups of all Campaigns.
  • We cannot add lists to only particular handpicked AdGroups
  • We cannot add lists to only of particular handpicked Campaigns.
  • We will not Pause/Activate any Campaigns/AdGroups.
  • We can add Remarketing Lists.
  • We can add Customer Lists.
  • We can add Remarketing Lists generated in Google Analytics, provided they abide by the RLSA policies.
  • We will not change the any existing bid modifiers on any existing lists
  • We will add RLSA on all AdGroups of all Search Campaigns and Shopping Campaigns(if any).
  • We can work on Search Network with Display Select Campaigns.
  • We will not be able to add a bid modifier if the Campaign/AdGroup is set to Auto-Bidding.
  • We will not implement RLSA with Target and Bid as the Targeting Setting.
  • We will only implement RLSA with Bid Only as the Targeting Setting.
  • If an AdGroup is set to Target and Bid as the Targeting Setting, then we do not add lists or bid modifiers to that AdGroup.
